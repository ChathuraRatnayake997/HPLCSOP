# HPLC Standard Operating Procedure (SOP)

A modern, web-based, and easily accessible guide for operating High-Performance Liquid Chromatography (HPLC) systems. This project aims to replace static, hard-to-update documents with a live, version-controlled, and user-friendly web page.

**Live Site: [https://chathuraratnayake997.github.io/HPLCSOP/](https://chathuraratnayake997.github.io/HPLCSOP/)**

### Key Features

*   **Digitally Accessible:** Access the SOP from any device with a web browser. No more searching for printed documents.
*   **Comprehensive Coverage:** Detailed sections cover everything from safety and setup to operation and troubleshooting.
*   **Easy Navigation:** A clean layout with a sidebar for quick navigation between sections.
*   **Searchable:** Use your browser's search function (Ctrl+F or Cmd+F) to find keywords instantly.
*   **Version Controlled:** Maintained on GitHub, ensuring that you are always viewing the most up-to-date, approved procedure.

## Who Is This For?

This guide is intended for:
*   **Laboratory Technicians & Analysts** performing routine HPLC analysis.
*   **Students & Trainees** learning about analytical chemistry and instrumentation.
*   **Researchers** who require a standardized procedure for their experiments.
*   **Quality Assurance (QA) Personnel** who oversee laboratory practices.

## Getting Started

### Viewing the Live SOP

The easiest way to use this guide is to visit the live website:
**[https://chathuraratnayake997.github.io/HPLCSOP/](https://chathuraratnayake997.github.io/HPLCSOP/)**

### Running Locally (for Developers/Contributors)

If you wish to modify or contribute to the project, you can run it locally on your machine.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/chathuraratnayake997/HPLCSOP.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd HPLCSOP
    ```
3.  **Open the `index.html` file in your favorite browser.**
    You can typically just double-click the file, and it will open.
